# Sleeplang-zZ
Uma linguagem de programação minimalista baseada em BrainF###k.
Modo de uso:
No terminal do sistema, use essa seguinte combinação de comandos:
python sleeptc.py <comando> <código entre aspas ou arquivo .slee>
Detalhe:
Os comandos disponíveis são:
exec/e - apenas executa o código
exec/d - executa e mostra o código em BrainF###k
exec/t - apenas mostra o código em BrainF###k
Comandos:
ZZZ: Incrementa.
zzz: Decrementa.
zzZ: Move para a célula da direita.
Zzz: Move para a célula da esquerda.
zZz: Saída (imprime o cáractere ASCII do valor atual).
ZzZ: Inicia um loop, mas se o valor da célula atual for zero ele pula para o comando após o fim do loop.
zZZ: Termina o loop, mas se o valor atual for diferente de zero reinicia o loop, se for zero ele continua para os próximos comandos.
